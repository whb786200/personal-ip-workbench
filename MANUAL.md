# 使用说明书

## 1. 打开系统

双击 `START-HERE.bat`。

系统会打开：

- 主工作台：http://127.0.0.1:8787
- 中文后台：http://127.0.0.1:8787/admin-cn.html
- 视频生成：http://127.0.0.1:8501

## 2. 中文后台怎么用

进入中文后台后，左侧有四个模块：

- 内容任务：记录选题、脚本、素材、成片、发布状态
- IP 画像：维护账号定位、人群、承诺、风格
- 素材库：保存链接、笔记、金句和参考素材
- 发布排期：规划公众号、小红书、抖音、视频号等内容

点击一行可以编辑；右上角“新增”可以添加记录；“同步到工作台”会把后台数据同步到主工作台。

## 3. 生成视频

打开 MoneyPrinterTurbo：

```text
http://127.0.0.1:8501
```

先在 `config.toml` 填好 DeepSeek 和 Pexels Key，然后在页面里输入主题或脚本生成视频。

## 4. 每日运营流程

1. 在“素材库”记录灵感和链接。
2. 在“内容任务”添加选题。
3. 用主工作台生成脚本提示词。
4. 到 MoneyPrinterTurbo 生成短视频。
5. 在“发布排期”安排平台和日期。
6. 发布后回填链接和状态。

## 5. 故障处理

运行自检：

```powershell
.\verify-workbench.ps1
```

如果中文后台打不开：

```powershell
.\run-workbench.ps1
```

如果 PocketBase 需要重新初始化：

```powershell
.\init-pocketbase.ps1
```
